# Image Assets

Place your optimized images here.

## Recommended Formats

- **WebP** — Best compression, modern browsers
- **PNG** — Fallback for older browsers
- **JPG** — For photos (not recommended for medical diagrams)

## Image Optimization

Use tools like:
- [Squoosh](https://squoosh.app/) — Online image optimizer
- [TinyPNG](https://tinypng.com/) — PNG/WebP compression
- ImageMagick — Command-line tool

## Suggested Images

1. **Hero image** (`hero.webp`) — Optional background or illustration
2. **Symptoms images** (`symptoms.webp`) — Visual representation of symptoms
3. **X-ray image** (`xray.webp`) — Example chest X-ray (with proper attribution)
4. **Transmission diagram** (`transmission.webp`) — How pneumonia spreads

## Usage in HTML

```html
<img src="assets/img/your-image.webp" 
     alt="Descriptive alt text" 
     loading="lazy"
     width="800"
     height="600">
```

**Important**: Always include descriptive `alt` text for accessibility.
